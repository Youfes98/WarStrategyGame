# Setup Guide

## Requirements

- [Godot 4.4+](https://godotengine.org/download)
- Python 3.10+
- Internet connection (for data pipeline)

## Step 1 — Install Python dependencies

```bash
pip install requests Pillow
```

## Step 2 — Run the data pipeline

```bash
cd tools
run_pipeline.bat        # Windows
# OR
python fetch_country_data.py && python geojson_to_godot.py
```

This downloads real-world data and generates:
- `data/countries.json` — all 195 countries with normalised gameplay values
- `data/adjacencies.json` — border graph for movement
- `assets/map/provinces.png` — 4096×2048 click-detection bitmap

## Step 3 — Open in Godot

1. Open Godot 4.4+
2. Click **Import** → select `C:\Claude\WarStrategyGame\project.godot`
3. Press **F5** to run

## Controls

| Input | Action |
|-------|--------|
| Space | Pause / Unpause |
| 1–5 | Set speed |
| Right-click drag | Pan camera |
| Scroll wheel | Zoom |
| Left-click country | Select |

## Project Structure

```
WarStrategyGame/
├── project.godot
├── scenes/
│   ├── Main.tscn          ← entry point
│   └── Map/Map.tscn
├── scripts/
│   ├── autoloads/         ← GameClock, GameState, UIManager, WorldMemoryDB, ProvinceDB
│   ├── map/               ← MapRenderer, MapCamera
│   └── ui/                ← SpeedControls, NotificationFeed, CountryCard
├── data/                  ← generated by pipeline
├── assets/map/            ← generated by pipeline
└── tools/                 ← Python pipeline scripts
```
